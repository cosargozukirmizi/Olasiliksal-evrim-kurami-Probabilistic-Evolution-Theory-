## SYNOPSIS

`BelowDiagonalTraversal::first(n)`

## DESCRIPTION

Returns the first object of the class. That is [1 2 ... n].

## RETURN VALUES

a list with integers from 1 up to n. 

## EXAMPLE

    >> package("BelowDiagonalTraversal"):
    >> BelowDiagonalTraversal::first(3);

                                                                       [1, 2, 3]
